# Supermarket Management System

这里是一个基于 ***SQL Server*** 和 ***C#*** 设计的一个管理系统，仅用于学习。

## 要求：
> - 根据课程设计任务，针对具体的实际应用进行需求分析，制定出数据库系统的设计方案。
> - 完成中小型数据库系统的开发、管理和维护。
    
## 课设选题与分组:
> - 课设选题：超市销售管理系统
> - 学生分组成立模拟开发项目组：（组长：付昱焯   组员：陈毅乐、冯铃凯）
> - 关系数据库系统开发过程及规范：选用 Microsoft SQL Server 进行数据库管理。 选用 C# 进行窗口UI开发。
    
## 数据库需求分析:
> - 概念结构设计
> - 视图设计
> - 数据库访问权限设计
> - 登录模块设计
